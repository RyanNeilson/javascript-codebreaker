A Mastermind Clone

How To Play:

When a user clicks the start button, the computer generates a random 4-color code with red, blue, green, yellow, black, and white as possibilities.

The user's task is to guess the code, which remains hidden until solved. *note: In this version of the game, colors will not be repeated in the generated code.

The user inputs a guess using the selector buttons to the right of the screen (or at the bottom on smaller, mobile devices). The buttons each correspond to the same-numbered column in the game board.

The user clicks "Submit" after choosing the four different colors he/she wishes to guess.

The submitted guess will appear in the first vacant row of the gameboard.

The last column of the gameboard will then display feedback related to the user's guess. The first number indicates the user has that number of colors correct, and those colors are correctly positioned on the board. The second number indicates the user has that number of colors correct, but those colors are incorrectly positioned on the board.

End of the game: The game is over when the user guesses the code correctly, or when the user has made 12 unsuccessful attempts at guessing the code.
